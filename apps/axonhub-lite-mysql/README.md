# AxonHub Lite MySQL

**AxonHub Lite** 是面向个人自托管场景的轻量版 AI Gateway。本应用使用 `ghcr.io/akuma-real/axonhub-lite` 镜像，并提供 MySQL 持久化配置，适合个人或小规模部署使用。

> 不知道啥原因，没法自动创建数据库，请先自行创建，然后填入对应的密码和用户名即可。

## 🚀 项目简介

AxonHub Lite 主要用于提供统一的 AI 服务接入层，简化多模型、多服务商的调用流程。平台兼容 **OpenAI、Anthropic** 等 API 格式，并支持请求记录、基础成本控制和负载均衡，帮助个人用户更方便地管理 AI API。

相比企业版，本版本更偏向个人轻量使用，弱化组织协作、团队治理等重型能力，部署和维护成本更低。

------

## ✨ 核心特性

### 🔗 统一 API 网关

- 完全兼容 **OpenAI / Anthropic** API 规范
- 自动将调用转换为多云、多模型提供商接口
- 通过 Transformer Pipeline 统一处理请求

### 🧵 全链路追踪（Tracing / Threads）

- 捕获每个请求的执行过程
- 支持线程化上下文
- 更适合调试、审计与性能分析

### 🔐 个人使用管理

- 适合个人 API Key 和渠道管理
- 保留基础访问控制和用量管理
- 避免企业组织协作功能带来的额外复杂度

### ⚖️ 智能负载均衡

- 多策略选择（权重、健康检查、会话一致性等）
- 自动选择最优模型/渠道
- 强化高可用性

### 🧩 综合开发工具

- 内置 Playground
- 项目管理与密钥管理
- API 调试体验优秀

------

## 🗄️ 个人轻量版说明

本应用使用 **AxonHub Lite 镜像** 和 **MySQL 数据库** 作为持久化存储，适用于：

- 个人自托管
- 小规模 AI API 网关
- 需要持久化请求记录和用量数据的场景

配置简单，无需额外手动初始化。

------

## 📦 部署说明

1. 通过 1Panel 安装应用
2. 配置 MySQL 连接参数
3. 启动容器后平台会自动初始化
4. 打开浏览器访问界面完成初始设置

支持 AMD64 / ARM64 架构。
