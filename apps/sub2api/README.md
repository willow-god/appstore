# Sub2API

Sub2API 是一个 AI API 网关平台，用于分发和管理 AI 产品订阅中的 API 额度。用户可以通过平台生成的 API Key 访问上游 AI 服务，平台负责认证、计费、负载均衡和请求转发。

## 功能

- 多账号管理：支持 OAuth、API Key 等上游账号类型
- API Key 分发：为用户生成和管理 API Key
- 精确计费：Token 级使用量统计和费用计算
- 智能调度：支持账号选择和粘性会话
- 并发控制：支持用户级和账号级并发限制
- 速率限制：支持请求和 Token 限流
- 内置支付：支持 EasyPay、支付宝、微信支付和 Stripe
- 管理后台：提供监控和管理 Web 界面

使用本项目可能违反 Anthropic 等上游服务商的服务条款。请在使用前阅读相关服务商的用户协议，并确保符合所在国家或地区的法律法规。因使用本项目导致的账号封禁、服务中断、数据损失或其他直接、间接损失，风险由使用者自行承担。
